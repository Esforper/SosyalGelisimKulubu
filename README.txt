
# websitesi

Sosyal Gelişim Klübü Web Sitesi README Dosyası 
#Proje Başlığı
Projemiz sosyal gelişim klübünün web sitesidir.Projenin aamacı web sitemiz üzerinden klüb tanıtımı, üye alımı ve  etkinliklerimizi duyurmaktır.

#Motivasyon

Sosyal Gelişim Klübü Web Sitesi olarak, topluma fayda sağlamayı hedefleyen bir platform oluşturmayı amaçlıyoruz. Bu web sitesi aracılığıyla, üyelerimizin kişisel ve sosyal gelişimlerine katkıda bulunmayı, bilgi paylaşımını teşvik etmeyi ve birlikte çalışarak olumlu değişimler yaratmayı hedefliyoruz.

Motivasyonumuz, insanların bir araya gelerek bilgi, deneyim ve fikirlerini paylaşmalarını sağlamak, toplum hizmetleriyle ilgili farkındalık oluşturmak ve insanların kendi potansiyellerini keşfetmelerine yardımcı olmaktır. Sosyal gelişim ve toplumsal sorumluluk bilincini ön planda tutarak, üyelerimize destek olmak ve toplumda olumlu bir etki yaratmak istiyoruz.

Bu web sitesi aracılığıyla, düzenlediğimiz etkinlikler, sosyal projeler ve toplumsal konularla ilgili haberler hakkında bilgilere erişebilirsiniz. Ayrıca, kulüp faaliyetlerimizi tanıtan içeriklere ve üyelerimizin deneyimlerine ulaşabilirsiniz.


Sosyal Gelişim Klübü Web Sitesi ekibine katılarak topluma fayda sağlama yolculuğunda bizimle birlikte hareket edin!

Bu bölüm, okuyucuya bu projeyi neden oluşturduğunuzu, böyle bir projeyi takip etmenin arkasındaki nedeni ve neden yapmaya karar verdiğinizi bildirmek içindir.

#Yapı Durumu

web sitesi tasarımı görüntü yüzdesi %125 olan bir cihazda yapılmasından dolayı web sitesi ile ilgili asıl verimi görüntü yakınlaştırma yüzdesi %125 olduğunda alınmaktadır.

#Kod Stili

kod stili olarak css kodları html kodlarının hemen üstünde yer almaktadır, belirli yerlerde html de artibüt ile tasarımsal eklemeler yapılmıştır (html ile iç içe tasarım da denebilir) o durumlar harici görseller tek görsel klasörü altında ayrı ayrı klasörler biçimindedir. tüm görsel çağırmalar o klasörden yapılmış olup internetten bir çağırma söz konusu değildir.

#Kurulum

Visual Studio Code'u açın.
Yeni bir HTML dosyası oluşturmak için CTRL + N (Windows) veya CMD + N (Mac) tuşlarına basın veya "Dosya" menüsünden "Yeni Dosya" seçeneğini seçin.
Yeni bir dosya oluşturulduğunda, dosya adını belirlemek için CTRL + S (Windows) veya CMD + S (Mac) tuşlarına basın veya "Dosya" menüsünden "Farklı Kaydet" seçeneğini seçin. Dosya adının sonuna ".html" uzantısını ekleyin (örneğin, "index.html").
Oluşturulan HTML dosyasının içine aşağıdaki temel HTML yapısını ekleyin:
html
Copy code:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Web Page</title>
</head>
<body>
    <h1>Merhaba, Dünya!</h1>
</body>
</html>
Yukarıdaki HTML kodunda <h1> etiketi içinde "Merhaba, Dünya!" metni görünecektir. Bu kısımları istediğiniz şekilde düzenleyebilir veya içerik ekleyebilirsiniz.

Dosyayı kaydetmek için CTRL + S (Windows) veya CMD + S (Mac) tuşlarına basın.
Artık Visual Studio Code'da HTML dosyanızı oluşturmuş oldunuz. Dosyayı tarayıcınızda açarak sonuçları görebilirsiniz.
  
web sitesini çalıştırmak için,
zip dosyası dışındaki klasörü vs code üzerinden ya da direkt ana_sayfa.html dosyasını çalıştırarak erişilebilir.

   ## Katkıda Bulunma

1. Fork yapın (https://github.com/kullaniciadi/repository-adi/fork)
2. Yeni bir branch oluşturun: `git checkout -b feature/ozellik`
3. Değişikliklerinizi yapın ve commit'leyin: `git commit -am 'Yeni bir özellik ekle'`
4. Branch'inizi uzak repository'ye pushlayın: `git push origin feature/ozellik`
5. Pull request açın
